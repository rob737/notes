Use case : Fetching all users fron the DB.

Each user can have multiple roles.

Authorization and Authentication :

JWT and OAuth2

Orchestration of the API :


@Entity
class User {

@Id
private int id;

@Column
private String name;


@ManyToMany(through = "roleId", junction="user_role")
private Role userRoles;

}

@Entity
class UserRole {

@Column
private String description;

@Column(mappedBy="User")
private User userId;


@Column(mappedBy="Role")
private Role roleId;


}



@Entity
class Role{

@Id
private int id;


@Column
private String roleName;

}
